Python Var Abbreviations
========================

Rule: As much as possible, abbrevs should be pronounceable rather than spelled out.

- var = variable
- abbrev = abbreviation
- num = number
- ins = instance
- arg = argument
- comp_key = composite key
- prop = property
- arg = argument
- conf = configuration
- db = database
    - Exception to the rule
- env = environment
- img = image
- info = information
- len = length
- max = maximum
- min = minimum
- msg = message
- param = parameter
- prev = previous
- proc = process
- req = request
- temp = temporary
- char = character
- klass = class
- lst = list
- doc = document | documentation
- id = identifier
- ob = object
    - Pronunced like in "job".
    - "obj" is hard to pronounce.
- gen = generate
- func = function
- ref = reference
